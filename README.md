digo-lang-zh-cn
===========================================
[digo](https://github.com/digojs/digo) 语言包：中文(简体)。

安装
-------------------------------
```
$ npm install digo-lang-zh-cn -g
```

用法
-------------------------------
### 命令行使用
```
$ tpack --dict tpack-lang-zh-cn
```

### 配置文件中使用
```js
tpack.dict = require("tpack-lang-zh-cn");
```
